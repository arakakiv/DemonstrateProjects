Frequently Asked Questions

Is it Open Source?

Yes, and u're allowed to contribute, look at the source code and all that our license allows.

How can I contribute?

Open an Issue in the official repository on github. Don't be shy!

What are keys?

They are passwords that must be pass on the request body to get all ur projects.

How to get my projects?

It's quite simple! Just do a GET request to our API! In the body, pass your key as json (key: value). If you wanna a single project, use, in query, an index parameter (&index=0). See the image below for more info!
